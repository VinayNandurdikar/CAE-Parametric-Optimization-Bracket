# Automated Parametric Optimization of an L-Bracket (ANSYS)## ObjectiveTo investigate the effect of rib placement on structural performance and identify the optimal configuration that minimizes both Total Deformation and Equivalent (von-Mises) Stress.---## Methodology### Geometry ParameterizationThe L-bracket geometry was parameterized in ANSYS DesignModeler by defining the rib’s horizontal position (H1) as an input parameter. This enabled automated geometry updates without manual redesign.### Simulation Setup- Analysis Type: Static Structural (2D Surface Body)- Mesh: Quadrilateral-dominant mesh- Boundary Conditions:  - Fixed Support applied to vertical flange  - Downward force applied on top edge### Parametric StudyA design table was created in ANSYS Workbench to evaluate multiple design points:- Input Parameter: Rib Position (H1) = 18 mm → 35 mm- Output Parameters:  - Equivalent Stress (MPa)  - Total Deformation (mm)---## Results![Stress Distribution](images/stress.png)![Design Table Results](images/graph.png)---## Key Findings- Rib placement significantly affects stiffness and stress distribution- Moving the rib closer to the load reduces bending moment- Stress redistribution observed with increasing rib distance- Optimal configuration achieved by balancing deformation and stress---## Engineering InsightThis study demonstrates how structural reinforcement placement impacts mechanical performance.Using parametric CAE workflows enables engineers to:- Explore multiple design configurations efficiently- Reduce manual effort- Make data-driven design decisions early in development---## Tools Used- ANSYS Workbench 19.2- DesignModeler- Static Structural (FEA)---## Skills Demonstrated- Parametric modeling- Automated simulation workflows- Design optimization thinking- Structural analysis interpretation
